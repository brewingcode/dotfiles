# Best: `assert_less`

Asserts one value is less than the other value.
This will exit the test immediately if it fails.



**Arguments:**

 - $1  `[string]`    -- The first value.
 - $2  `[string]`    -- The second value.



**Example:**

```bash
assert_less 1 2 # Success
assert_less 2 2 # Failure
```
