# Best: `assert_greater`

Asserts one value is greater than the other value.
This will exit the test immediately if it fails.



**Arguments:**

 - $1  `[string]`    -- The first value.
 - $2  `[string]`    -- The second value.



**Example:**

```bash
assert_greater 2 1 # Success
assert_greater 3 2 # Failure
```
