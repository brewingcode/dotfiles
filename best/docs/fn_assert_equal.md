# Best: `assert_equal`

Asserts one value equals another value.
This will exit the test immediately if it fails.



**Arguments:**

 - $1  `[string]`    -- The first value.
 - $2  `[string]`    -- The second value.



**Example:**

```bash
assert_equal 1 1        # Success
assert_equal "yes" "no" # Failure
```
